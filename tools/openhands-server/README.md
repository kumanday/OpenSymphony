# Pinned OpenHands Agent Server

This directory pins the local OpenHands SDK agent-server package used by OpenSymphony's local MVP.

- Package: `openhands-agent-server`
- Version: `1.14.0`
- Release date: `2026-03-13`
- Source: `https://github.com/OpenHands/software-agent-sdk/releases/tag/v1.14.0`

Use `uv sync` in this directory, then run `./run-local.sh`.
